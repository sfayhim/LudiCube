🎲 Cube éducatif interactif
Description

Cube ludique pour enfants 4-6 ans avec jeux éducatifs sur chaque face :

🔺 Formes : distinguer bonne/mauvaise santé

💡 Mémoire avec LED

🥗 Puzzle alimentation équilibrée avec feedback sonore

🫀 Corps humain : découvrir les organes

🎤 Assistant vocal : encouragements et questions

Objectifs

🧠 Développement cognitif et moteur

❤️ Bonnes habitudes de santé

📵 Réduire le temps d’écran

🎉 Apprentissage ludique et amusant

Contraintes

⚙️ Tech : Arduino, capteurs, modélisation 3D

🖥️ Son via PC (pas d’enceinte intégrée)

⏳ Temps : respecter les deadlines

IoT & Interactions

🔘 Bouton assistant vocal

🟢🔴 LED feedback sur pavé numérique

🫀 Boutons poussoirs corps humain

📀 Lecteur RFID

Utilisation

🏠 À domicile, sûr et transportable
